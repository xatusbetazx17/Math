Question 1
Solve the equation: √(6q − 5) = 5
Explanation
Square both sides: 6q − 5 = 25 ⇒ 6q = 30 ⇒ q = 5.
Question 2
Solve the equation: √(2k + 1) = 21
Explanation
Square both sides: 2k + 1 = 441 ⇒ 2k = 440 ⇒ k = 220.
Question 3
Solve the equation: √(2x − 7) − 5 = 0
Explanation
√(2x − 7) = 5 ⇒ 2x − 7 = 25 ⇒ 2x = 32 ⇒ x = 16.
Question 4
Solve the equation: √(7a − 2) − √(5a + 3) = 0
Explanation
Set radicals equal: √(7a − 2) = √(5a + 3). Square: 7a − 2 = 5a + 3 ⇒ 2a = 5 ⇒ a = 52.
Question 5
Solve the equation: √(p² − 3p + 32) = p + 2
Explanation
Square both sides: p² − 3p + 32 = (p + 2)² = p² + 4p + 4 ⇒ −3p + 32 = 4p + 4 ⇒ 7p = 28 ⇒ p = 4.
Question 6
Solve the equation: √(2x + 3) − √(x + 1) = 1
Explanation
Isolate one radical and square. You get x = −1 or x = 3, and both check in the original equation.
Question 7

Explanation

Question 11
Use the quadratic formula (all solutions are real): a² + 14a + 40 = 0
Explanation

Question 12
Use the quadratic formula (all solutions are real): x² = 5 − 8x
Explanation
Rewrite: x² + 8x − 5 = 0. Quadratic formula: x = (−8 ± √(64 + 20))/2 = (−8 ± √84)/2 = −4 ± √21.

Question 14
Find the slope of the line going through the points (−9, 0) and (0, −4).
Explanation
m = (−4 − 0)/(0 − (−9)) = −49.
Question 15
Find the slope of the line: −4y = 6 + 5x
Explanation
Solve for y: y = (6 + 5x)/(−4) = −(54)x − 32, so slope = −54.
Question 16
Identify the slope and y‑intercept of the line: f(x) = −6x − 9
Explanation
In y = mx + b form, m = −6 and b = −9 ⇒ y‑intercept is (0, −9).
Question 17
Graph the equation using the slope and y‑intercept: y = (25)x − 2. Choose the correct graph.
Explanation
The line must cross the y‑axis at −2 and rise 2 for every run of 5 (positive slope). That matches choice D.
Question 18
Write an equation of the line through (3, 5) with slope m = −47 (slope‑intercept form).
Explanation
Use y = mx + b with (3,5): 5 = (−47)·3 + b ⇒ b = 477, so y = (−47)x + 477.
